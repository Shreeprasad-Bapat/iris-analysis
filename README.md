# Exploratory Analysis of Iris Dataset in R

## Overview
This project demonstrates an exploratory data analysis workflow on a biological
dataset using R and the tidyverse. The focus is on feature engineering, grouped
summaries, and visualization.

## Tools Used
- R
- tidyverse (dplyr, ggplot2)

## Project Structure
- scripts/: R scripts for each analysis step
- plots/: generated visualizations

## Analysis Steps
1. Data loading and inspection
2. Feature engineering using ratio-based traits
3. Species-wise statistical summaries
4. Visualization of morphological separation

## Key Findings
- Petal measurements separate species more clearly than sepal measurements
- Setosa forms a distinct cluster
- Ratio-based features enhance inter-species separation

## How to Run
Run the scripts in numerical order:
1. 01_data_setup.R
2. 02_feature_engineering.R
3. 03_summary_analysis.R
4. 04_visualization.R
